# Embedded (`embedded/`)

Legacy Arduino/ESP32 firmware from the original Songbird hardware prototype era. These files target the **FeatherS2** microcontroller and are not part of the current macOS desktop DAW build.

## Files

| File | Purpose |
|------|---------|
| `songbird-chirp.ino` | Arduino sketch — main entry point for the FeatherS2 firmware. Handles OLED display, rotary encoders, MIDI I/O, and the sequencer loop. |
| `songbird.cpp` | Core Songbird music library — note sequencing, pattern playback, and MIDI output for the embedded platform. |
| `tests.cpp` | Unit tests for the embedded music library. |

## Hardware Platform

- FeatherS2 board by Unexpected Maker (ESP32-S2)
- FeatherWing OLED — 128x64
- MIDI FeatherWing
- 3x I2C QT Rotary Encoder
- DS3231 Precision RTC (optional)
- Quad Side-By-Side FeatherWing Kit

## Status

These files are **not actively maintained**. The current Songbird is a macOS desktop application built with JUCE + Tracktion Engine. See the [root README](../README.md) for the current architecture.

To build and flash the embedded firmware, see [`utils/README.md`](../utils/README.md) — the `setup.sh`, `flash.sh`, and partition tools target this hardware.
